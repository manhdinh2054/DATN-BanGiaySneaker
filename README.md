# Pansneaker - Spring Boot 

Dự án website bán giày trực tuyến được xây dựng trên nền tảng Spring Boot, tập trung vào sự tối giản, hiệu năng và trải nghiệm người dùng.
---
## ✨ Tính năng nổi bật.

🛒 Quản lý sản phẩm: Hiển thị danh sách giày, chi tiết sản phẩm, phân loại theo thương hiệu/size.

🔐 Bảo mật: Đăng nhập/Đăng ký với Spring Security (hỗ trợ JWT hoặc Session).

🛍️ Giỏ hàng & Thanh toán: Thêm sản phẩm vào giỏ, cập nhật số lượng và tạo đơn hàng.

🛠️ Trang Admin: Quản lý kho hàng, đơn hàng và thống kê doanh thu cơ bản.

🔍 Tìm kiếm: Bộ lọc thông minh theo tên, khoảng giá và danh mục.


## Công nghệ sử dụng

Backend: Java 17+, Spring Boot 3.x, Spring Data JPA, Spring Security.

Database: MySQL / PostgreSQL.

Frontend: Thymeleaf (hoặc React/Angular tùy dự án của bạn).

Build Tool: Maven / Gradle.
